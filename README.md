# Form Builder

A **dynamic form builder** built with **Next.js** that allows you to create, edit, reorder, and preview form fields in real-time. All form data is saved in the browser's **LocalStorage**, so your work persists across page reloads.

---

## Features

- Add multiple field types:
  - Text
  - Email
  - Number
  - Image upload
- Edit fields (label, placeholder, required)
- Delete fields
- Reorder fields via drag-and-drop (native HTML drag & drop)
- Live preview of the form
- Automatically saves to **LocalStorage**
- Fully **client-side**, no backend required

---

## Tech Stack

- **Frontend:** Next.js (React)
- **Styling:** Tailwind CSS (optional, can be plain CSS)
- **Storage:** Browser LocalStorage
- **No external libraries** required
